rpm工具使用
fgfgfgfg
   rpm  包管理
安装：
      rpm -i  <package_file>
  -i :安装
 -v:显示过程
 -h: 显示进度条

查询安装包信息：
      rpm -q [query-options]
 a:所有包信息
 f:指定文件有哪个程序生成
 i:包信息
 l:程序安装后所生成的文档
 c:程序的配置文件信息

卸载：
 rpm -e
 升级：
 rpm -uvh 

yum 工具

yum 配置文件信息：/etc/yum.repos.d/CentOS-Base.repo
配置yum :
[base]
name=centos#-localhost
baseURL=file:///mnt/cdrom   
gpgcheck=0/1 ：校验
 
yum 命令
yum repolist :显示已有yum仓库
yum list :显示yum程序包
yum install   :安装程序
yum update :升级包
yum check -update  :检查升级
yum remove : 卸载程序包
yum info :查看信息
yum clean : 清理缓存
yum deplist : 查看包依赖
yum history :yum 历史输入信息
yum -y ：默认 yes
yum -q :不显示过程

搭建本地光盘yum 源
mkdir /mnt/cdrom
mount /dev/cdrom /mnt/cdrom
creatrepo /mnt/cdrom
配置/etc/yum.repos.d/CentOS-Base.repo
